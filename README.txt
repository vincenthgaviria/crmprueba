Para poder reproducir esta app de manera local se deben seguir los siguientes requisitos:

1)tener descargado el programa xampp, con el cual se crea la base de datos en sql y se mantiene ejecutado en un servidor local.
2)ejecutar en xampp las opciones de Apache y Mysql.
3)en la opcion de Mysql, presionar el boton admin que nos redirecciona al navegador y a la central de administracion de las DB.
4)importar el archivo de DB llamado clientes.sql y con el mismo nombre.
5)abrir la ruta en el computador en donde se instalo xampp la cual es C:\xampp\htdocs.
6)Ubicado en la carpeta htdocs, crear una nueva carpeta con el nombre "Clientes" tal cual.
7)Dentro de esta carpeta Clientes pegar el archivo index.php.

8)ejecutar la terminal en la carpeta del proyecto.
9)ejecutar la aplicacion mediante el comando npm star.
